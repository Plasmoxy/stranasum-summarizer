STranASum

by Plasmoxy

Deployment of my Simple Transformer Abstractive Summarization model for my bachelor thesis. Provide input text with around same length as example. Click on example buttons to load cherry-picked example input texts, then press summarize to run model inference. In case of the Gigaword models, the maximum sentence length is 70 words. If you enter a longer text, the app will split the sentences/text into chunks and print multiple summaries.
