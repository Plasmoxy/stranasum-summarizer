Deployment of my Simple Transformer Abstractive Summarization (STranASum) model for my bachelor thesis, focusing specifically on summarization of news articles. Provide input text with around same length as example. Click on example buttons to load cherry-picked example input texts, or provide your own text, then press summarize to run model inference. The model is trained abstractive - it will attempt to understand and paraphrase the text. In case of the Gigaword models, the maximum sentence length is 70 words. If you enter a longer text, the app will split the sentences/text into chunks and print multiple chunk summaries. Each model output is capitalized and a dot is appended. Note that the models are not very large (~30M parameters) and is running on a CPU instance (for comparison, GPT-4 has ~175B parameters!) and may provide incorrect summaries.
